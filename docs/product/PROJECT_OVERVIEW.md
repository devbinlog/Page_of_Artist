# Project Overview — Page of Artist

## 한 줄 정의

아티스트가 자신의 음악을 3D 카드로 홍보하고, 팬이 장르 기반으로 새로운 아티스트를 발견하는 인터랙티브 플랫폼.

---

## 배경

기존 음악 플랫폼(Spotify, Apple Music)은 알고리즘 추천에 의존한다.
신인 아티스트는 알고리즘 진입 장벽으로 인해 노출 기회가 제한된다.
Page of Artist는 **시각적 탐색**을 통해 사용자가 능동적으로 새로운 아티스트를 발견하게 한다.

---

## 핵심 가치

| 가치 | 설명 |
|------|------|
| **시각적 발견** | 3D 카드 갤러리를 통한 직관적 탐색 |
| **아티스트 주도** | 아티스트가 직접 카드 페이지를 등록 |
| **장르 기반** | 취향에 맞는 장르 필터링 |
| **인터랙티브** | 카드 플립, 패럴랙스, 비닐 레코드 등 물리적 인터랙션 |

---

## 사용자 유형

### 팬 (Fan)
- 장르 갤러리를 탐색한다
- 카드를 클릭해 아티스트 정보를 확인한다
- Spotify / YouTube / Instagram으로 이동한다

### 아티스트 (Artist)
- 등록 페이지에서 자신의 카드 페이지를 만든다
- 앨범 아트, 트랙리스트, 소셜 링크를 등록한다
- 관리자 승인 후 갤러리에 노출된다

### 관리자 (Admin)
- 등록된 아티스트의 승인/거부를 결정한다
- 갤러리 품질을 유지한다
