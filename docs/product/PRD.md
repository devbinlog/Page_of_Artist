# Product Requirements Document (PRD)

## MVP 기능 범위

### P0 — 반드시 있어야 함

| 기능 | 설명 | 상태 |
|------|------|------|
| 3D 카드 갤러리 | 원형 캐러셀로 아티스트 카드 탐색 | ✅ 완료 |
| 카드 플립 | 앞면(앨범 아트) ↔ 뒷면(트랙리스트) | ✅ 완료 |
| 패럴랙스 틸트 | 마우스/자이로스코프 연동 카드 기울기 | ✅ 완료 |
| 장르 필터 | Pop / Hip-Hop / Rock / R&B 등 필터링 | ✅ 완료 |
| 아티스트 상세 페이지 | 카드 + 비닐 레코드 + 앨범 디스플레이 | ✅ 완료 |
| 아티스트 등록 | Firebase Firestore에 카드 데이터 저장 | ✅ 완료 |
| URL 라우팅 | React Router 딥링크 지원 | ✅ 완료 |

### P1 — 품질을 높이는 기능

| 기능 | 설명 | 상태 |
|------|------|------|
| 멀티스텝 등록 | 4단계 등록 플로우 | 🔲 미완 |
| Admin 페이지 | 등록 아티스트 승인/거부 | 🔲 미완 |
| 아티스트 description | 카드/상세 페이지 소개 텍스트 | 🔲 미완 |
| 디자인 토큰 | 색상/간격 중앙화 | 🔲 미완 |

### P2 — 고도화

| 기능 | 설명 | 상태 |
|------|------|------|
| 장르별 배경 테마 | 장르에 따라 씬 색상 변경 | 🔲 미완 |
| 음악 웨이브폼 시각화 | 재생 상태 시각화 | 🔲 미완 |
| 무한 스크롤 지원 | 갤러리 확장성 | 🔲 미완 |
| 모바일 최적화 | 터치 인터랙션 개선 | 🔲 미완 |

---

## 비기능 요구사항

| 항목 | 목표 |
|------|------|
| 초기 로딩 | 3초 이내 인트로 진입 |
| 갤러리 FPS | 60fps 유지 (데스크탑 기준) |
| Firebase 미설정 | 정적 데이터로 완전 동작 |
| 모바일 | 기본 탐색 가능 수준 |

---

## 제약사항

- Spotify API: Client Credentials Flow만 사용 (사용자 인증 없음)
- 아티스트 이미지: 외부 URL 입력 방식 (파일 업로드 미지원)
- Admin 인증: 간단한 비밀번호 기반 (JWT 불필요)
