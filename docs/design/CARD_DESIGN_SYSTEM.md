## 패럴랙스 레이어 구성

카드 앞면은 3개의 레이어로 구성된다.
각 레이어는 마우스 이동에 따라 다른 속도로 이동해 입체감을 만든다.

| 레이어 | depth | parallaxStrength | 내용 |
|--------|-------|-----------------|------|
| 0 | 0 | 0.008 | 아크릴 배경 (거의 안 움직임) |
| 1 | 1 | 0.016 | 프로그레스 바 |
| 2 | 2 | 0.022–0.026 | 텍스트, 플레이어 컨트롤 |

앨범 아트는 패럴랙스 레이어 밖에 z=0.05로 고정.
`renderOrder=10`으로 투명 배경 위에 확실히 렌더링.

## 인터랙션 규칙

### gallery 모드
```
첫 번째 클릭  → 전면 이동 + scale 0.65 → 1.0
더블클릭      → ArtistPage로 이동
플립          → 없음 (gallery에서는 플립 불가)
```

### detail 모드
```
클릭          → 플립 (앞/뒤 전환)
더블클릭 앞면 → GalleryPage로 돌아감
더블클릭 뒷면 → 앞면으로 되돌림
```

클릭/더블클릭 구분: 180ms 타이머로 싱글/더블 판별.


## 반응형 고려사항

현재 3D 카드는 데스크탑 기준으로 설계되었다.
모바일에서는:
- 마우스 틸트 → 자이로스코프 자동 전환 (`/Mobi|Android/i.test(navigator.userAgent)`)
- 스와이프로 캐러셀 이동 (`useSwipe` 훅)
- 카드 크기: Three.js 씬은 `fov` 기반으로 자동 스케일
